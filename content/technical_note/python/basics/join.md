### Basic join
Syntax: `str.join(iterable)`


```python
x = ["First", "Second", "Third"]
'_'.join(x)
```




    'First_Second_Third'




```python
num = '12345'
'-'.join(num)
```




    '1-2-3-4-5'




```python
let = 'abcdefg'
'.'.join(let)
```




    'a.b.c.d.e.f.g'
