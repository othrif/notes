# Jupyter shortcuts

Symbol         | Action
-------------- | -------------
y              | cell format -> code  
m              | cell format -> Markdown  
a              | Add cell above  
b              | Add cell below  
c              | copy cell  
v              | paste cell  
x              | cut cell  
d d            | delete cell  
z              | undo  
shift+z        | redo  
`<space><space>` | new line
`---`            | horizontal line
